# 작업 유형별 질문 뱅크

비개발자 친화 언어로 작성된 질문 후보. AI가 코드 검증 결과를 보고 옵션을 추측해서 채운다.

---

## 신규 (new)

### [3-2] 대상·범위
- "어떤 화면이나 기능에 추가돼요?"
  - 추측 옵션: 코드에서 보이는 후보 화면/기능들
- "기존 사용자에게도 보여요, 특정 조건일 때만 보여요?"

### [3-3] 시나리오
- 골든: "사용자가 [동작]하면 무엇이 일어나야 해요?"
- "결과는 바로 보여요, 어딘가에 알림이 와요?"
- 엣지: "[조건] 이런 경우엔 어떻게 동작해요?"
  - 추측 옵션: 안내 문구 / 빈 영역 / 비활성화 / 자동 숨김
- "잘못 입력하면 어떻게 알려줘요?"

---

## 옵션 추측 가이드

옵션을 추측할 때 우선순위:
1. **코드 검증 결과**에서 발견된 실제 후보 (실제 존재하는 화면, 컴포넌트, 라우트)
2. **기존 패턴**에서 유추 (비슷한 기능이 어떻게 동작하는지)
3. **일반적 UX 패턴** (안내 문구/비활성화/자동 숨김 등)

옵션은 항상 **구체적**이어야 한다. "방식 A vs 방식 B" 같은 추상적 라벨 금지.
